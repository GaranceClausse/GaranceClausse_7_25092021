Pour lancer l'application :  
-Ajouter un dossier images dans le backend  
-Ajouter un dossier .env dans le backend avec  
  DB_HOST=  
  DB_USER=  
  DB_PASSWORD=  
  DB_DATABASE=  
  RND_TKN=  
-npm install  
-node server.js dans le backend  
-npm run serve dans le frontend
